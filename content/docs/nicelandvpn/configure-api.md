You must be in Advanced Mode. To learn how to enable it, click [here](/docs/nicelandvpn/enable-advanced-mode).

## Step 1 Getting into the DNS settings
Select DNS from the sidebar.

## Step 2 Changing the IP and Port
In the Settings card, you can input the IP you want in the Server IP field and the port right below it in the Server Port field.

![](https://raw.githubusercontent.com/tunnels-is/media/master/nl-website/v2/sections/how-to-use-niceland/changing_api_ip_port.png)

## Step 3 Saving your changes
Click the SAVE button that appeared at the top to save your changes.
