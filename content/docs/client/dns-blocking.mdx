# Enable DNS over HTTPS for extra security

```md
1. Go to `DNS` in the main menu
2. Go to `Settings`
3. Enable `Secure DNS`
```

# Enable DNS blocking

```md
1. Go to `DNS` in the main menu
2. Go to `Block Lists`
3. Enable any block lists you want
```

# Bring Your Own List

```md
1. Go to `DNS` in the main menu
2. Go to `Block Lists`
3. Press `Create`
4. Enter a `URL` or `Tag`
```

:::tip

When a `URL` is not given for a blocklist, tunnels will look for that list in the `blocklists` directory using `Tag` as the file name

:::

# Create DNS Whitelists

Whitelists allow specific domains to bypass all blocklists. This is useful for domains that are incorrectly blocked.

```md
1. Go to `DNS` in the main menu
2. Go to `White Lists`
3. Press `Create`
4. Enter a `URL` or `Tag`
```

:::tip

When a `URL` is not given for a whitelist, tunnels will look for that list in the `whitelists` directory using `Tag` as the file name

:::
