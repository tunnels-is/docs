## Download
  Visit [the download page](https://tunnels.is/download) on the tunnels website and download the client for your platform.

## Run the client

### Linux
```bash
  # Tunnels will ask for network admin permissions the first time you run it.
  ./tunnels 
```

### MacOS
```bash
 sudo ./tunnels
```

### Windows
`Shift-RightClick` the `tunnels.exe` and select `Run As Administrator`

## Opening the UI
[Click here](https://localhost:7777) to open the UI.<br/>
URL: https://localhost:7777
